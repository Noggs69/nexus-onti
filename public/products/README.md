# Imágenes de Productos

## Estructura
Coloca las imágenes de productos en esta carpeta con nombres descriptivos:

```
public/products/
├── nexus-pro-x1.jpg
├── ultrabook-elite.jpg
├── canvas-tablet-pro.jpg
├── soundwave-ultra.jpg
├── nexus-lite.jpg
└── workstation-pro.jpg
```

## Formato recomendado
- **Formato**: JPG o PNG
- **Tamaño**: 800x800px (cuadrado)
- **Peso**: Máximo 200KB por imagen

## Uso en código
Las imágenes se referencian como: `/products/nombre-imagen.jpg`

Ejemplo:
```typescript
image_url: '/products/nexus-pro-x1.jpg'
```
